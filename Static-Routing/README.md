Static Routing Lab

In this project, I demonstrate  Cisco static routing configuration topology and ping test results.

The lab uses 4 routers, 4 switches, and 4 PCs connected in a square topology. Each router connects to its own LAN and to neighboring routers.

Static routes were configured on each router to allow communication between all LAN networks.

Tools: Cisco Packet Tracer
Technologies: Static Routing, IPv4 addressing

Result: All PCs can successfully communicate across the network.
