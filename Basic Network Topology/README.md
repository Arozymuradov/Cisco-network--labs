Basic Two-Routers Network Lab

In this lab, I demonstrate how two separate Local Area Networks (LANs) can communicate with each other through routers. 
Two Cisco ISR4331 routers are connected using a point-to-point network and each router connects to a switch that serves multiple hosts. 
The purpose of this lab is to configure IP addressing and static routing so that devices in both networks can successfully communicate. 

Skills: Static routing, network design, IP addressing, troubleshooting 

Tools: PUTTY, Hardware - Switch, Router, PC, Cisco Packet Tracer, Cisco IOS CLI.
